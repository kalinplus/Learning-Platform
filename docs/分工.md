- 第一列"负责人"待各成员认领后填写。

## 各模块详细说明

### 1. 项目仓库 + 技术架构 + 审核 PR

- 制定 Git 分支规范与 PR 流程
- 确定前后端技术选型并搭建基础框架
- 配置开发/测试环境，定义接口规范
- 审核团队成员的代码提交，合并至主分支

### 2. 项目文档 + 全局协调

- 维护需求文档、架构文档、接口文档
- 编写测试用例，协助各模块联调
- 定期汇总项目进度，同步团队风险
- 统一代码命名规范与开发标准

### 3. 知识图谱构建（数据层）

- 将《数据结构》各模块知识点整理入库
- Neo4j 图数据库的节点与关系导入
- 配置知识点层级关系（如"线性表"包含"栈"）与关联关系（如"栈"与"递归"）
- 维护知识点属性（定义、核心考点、难度等级等）

### 4. 知识图谱交互式可视化

- 使用 ECharts/D3.js 实现节点-边图谱展示
- 支持图谱缩放、平移、旋转、搜索、高亮
- 根据用户学习进度标记节点状态（已掌握/未掌握）
- 适配 PC 端与移动端屏幕

### 5. 学习目录功能

- 基于知识图谱层级关系生成多级目录
- 同步用户学习状态（未学习/学习中/已完成）
- 根据知识点依赖关系推荐学习路径
- 支持目录折叠/展开交互

### 6. 知识点详细讲解（图文）

- 开发知识点详情页：定义、图示、操作步骤
- 复杂度分析页面（时间/空间复杂度、最优/最坏/平均情况）
- 内容排版与展示优化
- 配套练习题入口

### 7. 代码实现与在线运行

- C++ 代码展示组件（语法高亮、行号、一键复制）
- 代码在线运行/调试功能
- 正常场景与异常场景的代码示例
- 代码执行结果与错误信息展示

### 8. 习题模块

- 题库录入（选择题、填空题、编程题）
- 答题交互与自动判分
- 错题记录与复习功能
- 答案与详细解析展示

### 9. 学习跟踪与统计分析

- 记录用户学习行为（知识点、时长、视频进度、做题情况）
- 统计整体学习进度与模块完成率
- 生成可视化图表（柱状图、折线图、雷达图、热力图）
- 学习报告生成与导出

### 10. AI 辅助功能

- 对接 AI 问答接口，支持多轮对话
- 代码分析与纠错（语法检查、逻辑分析、优化建议）
- 代码生成（根据需求生成 C++ 代码）
- AI 对话上下文管理

### 11. 视频讲解 + 全局整合

- 视频播放模块（支持进度记录）
- 跨模块联调与数据接口对接
- UI 风格统一优化
- 整体功能体验测试与 Bug 修复
